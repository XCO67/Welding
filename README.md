# Welding Company Portfolio Website

A modern, clean portfolio website for a welding company showcasing services and work.

## Features

- Clean and modern design
- Responsive layout
- Smooth animations
- Services showcase
- Portfolio gallery
- About section

## Services Offered

- CNC Machining
- Sheet Metal
- Laser Cutting
- Acrylic
- Powder Coating
- 3D Printing
- Engraving

## Technologies

- HTML5
- CSS3
- JavaScript

## Deployment

This website is ready to be deployed on Vercel.
